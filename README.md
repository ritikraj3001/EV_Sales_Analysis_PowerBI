This project provides valuable business insights through dynamic KPIs and interactive visualizations, enabling stakeholders to make data-driven decisions.

# 📌 Project Overview

The **EV Sales Analysis Dashboard** provides a comprehensive overview of the electric vehicle market by tracking revenue, sales performance, top brands, vehicle models, drive types, and yearly sales trends. The dashboard is fully interactive, allowing users to filter data by **Year, Country, and Brand**.

---

# 🎯 Objectives

- Analyze overall EV sales performance.
- Monitor total revenue and sales.
- Identify top-performing brands and vehicle models.
- Compare drive types and body types.
- Track yearly sales growth.
- Enable interactive business analysis through slicers.

## 🧮 Analytical SQL Queries

### Create sql Database
```sql
create database sql_project;
```

### craete Table
```sql
Create table ev_vehicles(
brand varchar(50),
model varchar(50),
years int,	
variant	Varchar (50),
price_usd int,	
drive_type varchar(50),
seating_capacity int,	
body_type varchar (20),	
weight_kg int,
safety_rating int,	
country varchar(50),
segment varchar (50),	
annual_sales int
);
```

### See the created Table
```sql
Select * 
from ev_vehicles;
```

## Imported data EV_market from csv to pgadmin into ev_vehicles table

### To count how many rows into table
```sql
select count(*) 
from ev_vehicles;
```


### Total Brand
```sql
select distinct brand 
from ev_vehicles;
```


### Total Model
```sql
select distinct model 
from ev_vehicles;
```


### How many models in each brand
```sql
select brand, Count(model) as model 
from ev_vehicles
group by brand;
```


### Total sales
```sql
select sum(annual_sales) 
from ev_vehicles;
```


### sales by each year
```sql
select years,sum(annual_sales) as total_sales 
from ev_vehicles
group by years
order by years;
```


### sales by each brand
```sql
select brand,sum(annual_sales) as total_sales 
from ev_vehicles
group by brand
order by Total_sales desc;
```


### Total Revenue generated
```sql
select sum(price_usd*annual_sales::bigint) as Revenue 
from ev_vehicles;
``` 


### Total Revenue generated by each brand
```sql
select brand, sum(price_usd*annual_sales::bigint) as Revenue 
from ev_vehicles 
group by brand
order by revenue desc;
```


### Revenue by each year
```sql
select Years, sum(price_usd*annual_sales::bigint) as Revenue 
from ev_vehicles 
group by years
order by years;
```


### Top 5 Revenue genetated brand
```sql
select brand, sum(price_usd*annual_sales::bigint) as Revenue 
from ev_vehicles
group by brand
order by revenue desc
limit 5;
```


### Top 5 Revenue genetated country
```sql
select country, sum(price_usd*annual_sales::bigint) as Revenue 
from ev_vehicles
group by country
order by revenue desc
limit 5;
```


### Average sales in years
```sql
select years, round(avg(annual_sales),0) from ev_vehicles
group by years
order by years;
```


### Maximum and Minimum sales
```sql
select max(annual_sales)as Max_sale, min(annual_sales) as Min_sale
from ev_vehicles;
```

# 💡 Key Business Insights

- 🚗 Tesla dominates EV revenue among all brands.
- 📈 EV sales experienced strong year-over-year growth from 2020 to 2025.
- 📉 Sales declined in 2026, indicating incomplete data.
- 🏆 Model Y is the highest revenue-generating EV model.
- ⚙️ AWD vehicles account for the largest share of total sales.
- 🚚 Trucks contribute the highest revenue among all body types.
- 🌍 The dashboard supports country-level analysis through interactive filters.
